# SOL25 Language Toolchain

## May 2025

### Description

This project implements a complete toolchain for the SOL25 object-oriented
programming language, consisting of a Python parser and a PHP interpreter.

#### 1. Parser `parse.py`

The parser reads SOL25 source code from standard input and outputs an XML
representation of the Abstract Syntax Tree (AST) to standard output.

#### Parser usage

```sh
python parse.py < input.src > output.xml
```

Options:

- -h, --help: Displays usage information.

#### 2. Interpreter `interpret.php`

The interpreter reads the XML AST generated by the parser and executes the
program logic.

#### Interpreter usage

```sh
php interpret.php --source=output.xml [options]
```

Options:

- --source=file: Path to the XML file generated by the parser.
- --input=file: Path to a file containing inputs for the interpreted program
  (optional).
- --help: Displays usage information.
